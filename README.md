# Productivity Suite (Python Tkinter)

## Preview

![App Screenshot](screenshot.png)

A multi-tool desktop productivity application built using Python and Tkinter.

## Features

- Alarm Clock  
- Pomodoro Timer  
- Digital Clock  
- To-Do List with file storage  
- Dark Mode UI  
- Tab-based interface  

## Technologies Used

- Python  
- Tkinter  
- Git & GitHub  

## How to Run

```bash
python productivity_app.py
